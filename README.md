# 📊 Data Analysis on Indian Commodity Market

## 📌 Project Overview
This project analyzes historical data of the Indian commodity market to uncover trends in **price movements, trading volumes, and market behavior**. It focuses on major commodities such as Gold, Silver, Crude Oil, and Copper.

The goal is to transform raw data into meaningful insights using data analysis and visualization techniques.

---

## 🎯 Objectives
- Analyze **Open, High, Low, Close (OHLC)** prices
- Study **trading volume patterns**
- Identify **yearly and monthly trends**
- Compare **commodity performance**
- Visualize insights using graphs and charts

---

## 📂 Dataset Information
- **File Name:** `commodities_indian_units.csv`
- **Features:**
  - Date
  - Commodity
  - Open Price
  - High Price
  - Low Price
  - Close Price
  - Volume

## ⚙️ Workflow

### 🔹 Data Collection
- Dataset collected from public sources like Kaggle

### 🔹 Data Preprocessing
- Handling missing values
- Removing outliers
- Converting data types
- Feature Engineering:
  - Extract Year & Month
  - Price Range = High - Low
  - Daily Change = Close - Open

### 🔹 Data Analysis
- Average Close price per commodity
- Total trading volume
- Maximum High & Minimum Low values
- Year-wise and month-wise trends
- Commodity comparisons

### 🔹 Data Visualization
- Bar Charts
- Line Charts
- Scatter Plots
- Box Plots
- Heatmaps

---

## 📈 Key Insights
- Gold and Silver show steady long-term growth
- Crude Oil is highly volatile
- Strong relationship between Open and Close prices
- Trading volumes increase during market uncertainty

---

## ⚠️ Limitations
- No real-time data
- External factors (news, politics) not considered
- Currency fluctuations not included
